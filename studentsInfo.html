Краткий справочник по необходимым функциям

document.getElementById(id);
Возвращает ссылку на элемент по его идентификатору (ID);
id идентификатор является строкой, которая может быть использована для идентификации элемента;
setTimeout(func, delay)
Позволяет задавать вызов функции через заданный период времени.
func Функция для исполнения через заданную задержку.
delay Задержка в миллисекундах, 1000 миллисекунд равны 1 секунде.
element.addEventListener(event, handler);
Позволяет назначит обработчик события для элемента
event Имя события, например click
handler Функция обработки
setInterval(func, delay)
Позволяет задавать повторяющийся вызов функции через заданный интервал времени.
func Функция для исполнения через заданную задержку.
delay Задержка в миллисекундах, 1000 миллисекунд равны 1 секунде.
element.innerText
Своство DOM элемента позволяющее получить/задать текст внутри элемента
element.clientWidth
Своство DOM элемента в котором содержится ширина элемента
element.clientWidth
Своство DOM элемента в котором содержится высота элемента
canvas.drawImage(img, x, y, width, height)
Вывод изображения в заданной позиции на холсте
img Изображение для рисования
x Координата X на холсте, куда будет помещено изображение
y Координата Y на холсте, куда будет помещено изображение
width Применяемая ширина изображения
height Применяемая высота изображения

GameController API

gameController.icons
Объект с картинками для иконок
gameController.user
Объекст с информацие о текщем пользователе
Содержит следующие поля:
id идентификатор игрока;
isAdmin является ли пользователь администратором;
login логин пользователя;
nativeName имя пользователя;
role роль пользователя;
gameController.game
Объекст с информацие о текущей игре
Содержит следующие поля:
id идентификатор игры;
owner объект с информацией о пользователе, создавшем игру;
team1Stats объект с информацией о 1 команде(оставшееся количество жизней, набранные очки, ...);
team2Stats объект с информацией о 2 команде(оставшееся количество жизней, набранные очки, ...);
switchTimeout
gameController.log
Функция для логирования, необходимо написать самостоятельно
gameController.incrementProgress
функция для изменения состояния progressBar, необходимо написать самостоятельно
gameController.remainingSwitchTime
время, оставшееся до смены ролей в миллисекундах
gameController.displayPlayers
Функция рисования игрока, необходимо написать самостоятельно
gameController.displayMap
Функция рисования карты, необходимо написать самостоятельно
